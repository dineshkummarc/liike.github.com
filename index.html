Project Liike (LEEE-keh)

Movements in mobile web application development (sounds serious, huh?)

  • RSS
  • Blog
  • Archives

Project Liike is a effort out of the Microsoft patterns & practices team to help developers build mobile web applications for modern devices.

Right now we’re still just getting the project off the ground, but we’ll be adding more details over the coming weeks. If you’re interested in this space, be sure and stay tuned.

Liike is a Finnish word that means motion/movement, which not only represents how the web moving from desktops to devices, but also the mobility the devices provide. Also has a nice ring to it :)
